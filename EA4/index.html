Parametrisierte Flächen

1. Pillow Shape

x = cos(u)
y = cos(v)
z = a · sin(u) · sin(v)

u ∈ [0, π], v ∈ [-π, π]

2. Möbius Band

x = [R + s·cos(t/2)]·cos(t)
y = [R + s·cos(t/2)]·sin(t)
z = s·sin(t/2)

s ∈ [-w, w], t ∈ [0, 2π]

3. Toruskissen (Eigene Parametrisierung)

x = [R + r·cos(v)]·cos(u)
y = [R + r·cos(v)]·sin(u)
z = r·sin(v) + a·sin(u)·sin(v)

u, v ∈ [0, 2π]

(Kombination: Torus-Grundform + Pillow-Shape Ausbeulung)
